Previous Work

Conference Department Portal

Another internal resource project. This was a solo project that was created 100% from scratch as originally their record keeping was only done on paper.

  • Setup an XAMPP server for the department.
  • Utilized proper associated SQL tables for storing all of their required information and files on their server.
  • Front end uses Bootstrap 4 to style everything in a responsive, format appearance.
  • Utilized Jquery and Ajax to send data from the front end to PHP pages for processing before insertion or editing of the databases.
  • Properly stripped and secured all data before sending to databases. (Security is important when dealing with data, a businesses #1 resource.)
  • Included a jQuery map that allows the user to click on a state and display all conferences that are scheduled in that state.
  • Implemented a calander that displays conferences from the database on their corresponding dates so the department can edit them or make quick changes.
  • Implemented a Javascript drag and drop utility for uploading files to the server for each conference, storing them in labled folders for each one.
